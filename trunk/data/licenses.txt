This file contains license and copyright information for the Ultimate Stunts data files. For every file, the following things are listed:

Original author:
	the original author of the data

Original project:
	the project (software / website / ...) where the data was taken from. For some files, the data was not taken anywhere, but created "from scratch" for Ultimate Stunts. In these cases, Ultimate Stunts is listed as the original project.

Original license:
	the license of the original data.

UStunts author:
	the person who included this file in Ultimate Stunts. Not listed when the original project is Ultimate Stunts.

UStunts modifications:
	modifications made to the file when importing it into Ultimate Stunts. Not listed when the original project is Ultimate Stunts.

As the task of listing these things started long after the first data files were created, it is extremely hard to find this legal information. Therefore, this information is incomplete. The intention of creating this file is to make this information more complete by collecting it and writing it down.

For the names of Ultimate Stunts developers, the following abbreviations are used:
	CJP:    	Corné Plooy
	BBalazs:	Borbola Balazs

In any case, Ultimate Stunts releases these files under a GPL-compatible license, which means that you can use them as a part of GPL-licensed packages. If it is discovered that this is not allowed by the original license, either the original author will be contacted to ask for permission, or the file will be replaced/removed.
