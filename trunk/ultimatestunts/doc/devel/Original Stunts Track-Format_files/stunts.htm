Stunts Trackviewer
 
Ein in php4 (mit gd) geschreibener Trackviewer für das Kultspiel Stunts (4D Sports Driving) aus den frühen 90'er Jahren des vorherigen Jahrhunderts.
Die Software läuft auf einem Webserver und setzt eine php-instalaltion (>=4.1 mit gd) vorraus. Der Track (*.trk) wird per HTTP hochgeladen und das daraus generierte Bild (.png) an den Client geschickt. Jeder aktuelle Browser kann dieses Format darstellen.
Ebenfalls enthalten ist eine Beschreibung des Streckenformates.
